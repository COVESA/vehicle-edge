# Prerequisites

The contents of the folder _./config_ has to contain the following directory structure

```code
certs
|- jwt.key.pub
|- Server.key
L- Server.pem
vss.json
```

You can find out, where to get these files [here](https://github.com/GENIVI/iot-event-analytics/docker/vss2iotea/README.md)
